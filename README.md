# URL
[AI-SCOPE](https://aiscopeseoul.com/)

# Description
위 궤양에는 3가지 종류(BGU, EGC, AGC)가 있다.

사용자가 궤양의 사진을 드레그나 select file 을 하여 궤양의 위치를 파악해 궤양의 종류를 파악해 표출해 준다.

# Stack
React.js, css
